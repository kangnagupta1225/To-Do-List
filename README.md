To-Do List

A modern and responsive To-Do List web application built using **HTML5**, **CSS3**, and **Vanilla JavaScript**. This project helps users manage daily tasks efficiently with a clean user interface, local storage support, and useful productivity features.

Features

- Add new tasks
- Mark tasks as completed
- Edit existing tasks
- Delete individual tasks
- Clear all completed tasks
- Task filtering (All, Active, Completed)
- Search tasks instantly
- Light and Dark mode
- Responsive design for desktop and mobile
- Data stored using Local Storage
- Smooth animations and modern UI

Built With

- HTML5
- CSS3
- JavaScript (ES6+)
- Local Storage API

How to Run

1. Clone the repository

git clone https://github.com/kangnasync/To-Do-List.git


2. Navigate to the project folder

cd To-Do-List

3. Open `index.html` in your browser.

Alternatively, use the Live Server extension in Visual Studio Code for the best experience.
